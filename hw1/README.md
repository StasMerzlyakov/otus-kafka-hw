# kafka run

### Примечания

Перед запуском тестов выполнить:
- ./get-kafka.sh

- через конфигурацию kafka1.yml из репозитория OTUS-kafka (start-compose.sh)
- через собственные контейнеры (build-images.sh; start-kafka.sh/stop-kafka.sh)


Работа с топиком:
- ./create-topic.sh
- ./write-to-topic.sh
- ./read-from-topic.sh
